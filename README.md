# Ajanta Photography Frontend

React + Vite frontend that uses Supabase as the backend platform.

## Tech stack

- React
- TypeScript
- Vite
- Tailwind CSS
- Supabase (Auth, Database, Edge Functions, Storage)

## Local development

1. Install dependencies:

```sh
npm install
```

2. Create/update `.env` with:

```sh
VITE_SUPABASE_URL=your_supabase_url
VITE_SUPABASE_ANON_KEY=your_supabase_anon_key
```

3. Run the dev server:

```sh
npm run dev
```
